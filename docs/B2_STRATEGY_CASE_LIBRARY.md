## 案例 1: 星环科技 (688031)

### 基本信息
- **代码**: 688031
- **名称**: 星环科技
- **板块**: 数据库软件 / 大数据 / 国产替代
 - **分类**: 横盘突破

### 关键日期
- **B1 触发日**: 2025-12-04（实盘：J=-4.05，趋势线上方，属于低位触发）
- **B2 突破日**: 2025-12-05（实盘：收盘 72.96，涨幅 +17.02%，成交量 1626 万手，vol/10dAvg=3.02x）

### 选择理由
1. **行业赛道优质**: 国产数据库软件，受益于信创政策长期红利，基本面支撑估值，主力有做多动力。
2. **技术形态教科书级**: 整个 B2 形成过程清晰，每个步骤均有明确的量价形态支撑，重现性强，便于作为模板推广扫描。
3. **六步逻辑严丝合缝**:
   - **盘整区间**: 2025-10-31至 2025-12-04，区间高低点稳定，量能萎缩，换手率低（实盘收盘 64.95 → 62.35，区间震荡于 62–67 元）。
   - **攻击波**: 2025-12-05（收盘 72.96，单日+16.2%）放量突破区间价格
   - **B1 触发日**: 2025-12-04，收盘 62.35，KDJ 的 J=-4.05（<13，且满足放宽阈值<20），短期趋势线 > 多空线。
   - **B2 突破日**: 2025-12-05，收盘 72.96，涨幅 +17.02%，成交量 1626 万手（vol/10dAvg=3.02x），J=56.77，全面满足 B2 突破条件。
   - **关键顺序约束**: B2 买点前一交易日必须是 B1（即 12-04 -> 12-05），避免“跳日触发”带来的误选。
4. **止损逻辑清晰**: B2 突破日最低价作为硬止损位，破位即离场，控制最大回撤。

## 案例 2: 晶科科技 (601778)

### 基本信息
- **代码**: 601778
- **名称**: 晶科科技
- **板块**: 光伏 / 新能源（示例）
- **分类**: 灾后重建

### 关键日期
- **B1 触发日（反转标志）**: 2025-06-17
- **B2 突破日**: 2025-08-07

### 选择理由
1. **事件背景**: 公司先经历大幅回调或利空（“灾后”），随后出现放量回补与经营/市场改善的信号，适合用 B2 捕捉趋势修复初期的强势反弹。
2. **关键节点（本地 CSV 可复核）**:
   - **2025-06-17**：放量大涨，作为反转的标志性节点（需回溯计算成交量与近10日均量比值）。
   - **2025-08-07**：突破日，收盘 > 整理区间高点且涨幅 >= 4%，成交量同时放大，满足 B2 的放量突破条件；前一日 KDJ 的 J 值处于低位（回溯时应计算并验证）。
3. **策略适用性**: 灾后修复场景下，主力回补与情绪修复常伴随放量突破，B2 可在确认放量与 KDJ 条件下提供可操作的入场点。

### 复盘要点
- 验证指标：B2 当日 KDJ 中 J 值 < 60；前一交易日 J < 20；成交量 > 近10日均量 x 1.5；收盘价突破整理区间高点且涨幅 >= 4%。
- 建议回测区间：2025-06 至 2025-09，重点核验止损位与持仓持续性。

### 灾后重建注释
- 市场行为解读：在击穿对手盘（短期趋势线之上买入的全死，短线入场的全给他埋了）之后，随即出现一根巨量长阳反包。
   这类巨量长阳反包通常是主力强力介入的信号，表明前期被套或抛压被吸收，随后主力开始回补筹码并推动股价向上。回测时应关注当日与前几日的量价配合以及是否存在异常换手。

## 案例 3: 四会富士

### 基本信息
- **代码/名称**: 四会富士（300852）
- **分类**: 平行重炮

### 含义（定义性描述）
- 股价在相对平行位置出现两根或多根起放量的长阳（大阳），这些大阳的收盘价处于相近价格区间（±2%）。
- 在形成上述大阳前后，股价处于一个 10–20 天的整理平台区间；在 B2 日通常可见 2–4 根大阳（优选光头阳线，单日涨幅 >6% 为最佳），其间夹杂缩量的小阳线/小阴线，整体表现为“红肥绿瘦”（阳线多且实体大，阴线少且实体小）。
- 大阳之间的阴线成交量显著小于大阳的成交量，且一般不突破多空/知行线的关键支撑。

### 可量化的判定要点（用于回溯/筛选）
- 平台期长度：10–20 个交易日，区间振幅较小，成交量整体萎缩。
- 存在至少 2 根放量大阳，收盘价处于相近区间（相差 ≤ ±2%）。
- B2 突破日前后 3 日内出现累计 2–4 根大阳（其中至少 1 根为放量 >= 6% 的光头阳更佳）。
- 在大阳与大阳之间的阴线/小阳线，成交量 <= 最近大阳成交量的 30%（或显著小于近 5 个大阳的平均量）。
- B2 日的 KDJ：当日 `J < 60`，且**前一日** `J < 20`（该条件用于排除已过度超买的反弹）。
- 放量确认：B2 日成交量 > 近 10 日均量 × 1.5；收盘价突破整理区间并涨幅 >= 4%。

### 行为特征（形态描述）
- 多根中小阴线/缩量小阳置于两根或多根大阳之间，形成“2 阳夹数阴”的局面；整体 K 线序列表现为红线粗大、绿线细小。
- 关键支撑（多空线/知行线）在阳线之间没有被明显破坏，说明主力控制节奏、分批吸筹。

### 示例（用户提供的观察）
- 四会富士在 2025-08-20、2025-08-28（及临近日）出现了多根大于 4% 的大阳线；在 2025-09-09 的 KDJ 小于 13 之后，第二天出现放量大阳（上涨 4.98%），随后第三天涨停。该序列符合“平行重炮”的量价与节奏特征。

**对应收盘价（本地 CSV: 300852，数据已核实）**
- 2025-08-20: 36.89（涨幅 +3.54%，量/10日均量=1.98x）
- 2025-08-28: 36.23（涨幅 +6.50%，量/10日均量=1.67x）
- 2025-09-09: 35.17（缩量回踩日，量/10日均量=0.44x，B1 低 J 值候选日）
- 2025-09-10: 36.91（B2 日，涨幅 +4.95%，量/前日=1.87x；注：此日量/10日均量=0.84x，低于通用1.5x，系前期大量日拉高均量所致，以量/前日倍数辅助判断）

### 基本信息
- **代码**: 002779
- **名称**: 中坚科技
- **分类**: 平行重炮（Case3 的同类子类）

### 关键时间线（用户提供观察）
- **底部确认日（放量大阳/涨停）**: 2025-06-11（该日为放量大阳/涨停，确认了低位区间与底部价位）
- **平行区间放量大阳期**: 2025-07-25 至 2025-08-01（在相对平行的价格区间内出现 2 根及以上放量长阳）
   - **第一根大阳**: 2025-07-25，收盘上涨约 5.8%（放量）
   - 缩量回调 4–5 个交易日后出现 B1（且站上知行/多空线）并确认随后出现 **大阳 +5.57%（放量 >= 前一日 2 倍）**
   - 随后次日继续拉升 **+7.83%**，确认突破并完全打破平行区间

### 分类理由与要点说明
1. **底部确认**：2025-06-11 的放量大阳/涨停作为底部与价格区间确认点，具备主力先行介入的典型特征。
2. **平行重炮特征**：在 25–30 个交易日后，于一个相对平行的价格区间内出现多根放量大阳，且这些大阳的收盘价位置相近（文档化判定时建议使用 相差 ≤ ±2% 作为量化近似条件）。
3. **节奏与量能**：第一根大阳（2025-07-25）与随后确认大阳（+5.57%）之间有 4–5 日的缩量回调，回调不破关键支撑（多空/知行线），符合分批吸筹与主力节奏控制的行为学特征。
4. **放量强确认**：确认大阳当日成交量 >= 前一日成交量 × 2（用户说明），且随后一日继续放量并大幅拉升（+7.83%），完成对平行区间的实质性突破。

### 可量化回溯检查项（便于脚本验证）
- 底部日（2025-06-11）成交量 vs 近 10 日均量（放量程度）
- 2025-07-25 与后续大阳（+5.57%）的收盘价差异（建议阈值：≤ ±2%）
- 确认大阳当日成交量 / 前一日成交量 >= 2
- 确认大阳当日成交量 > 近 10 日均量 × 1.5
- B1 出现时 `短期趋势线 > 多空线` 且 B1 前一日 `J < 20`，确认日 `J < 60`
- 回溯时记录：每个关键日的 KDJ(J)、当日成交量、近 10 日均量、以及大阳前后 5 日的量比（volume ratios）

## 策略逻辑总览
1. **B1 前提检查**: 短期趋势线 > 多空线，J 值低位 (<13)，价格贴近短期趋势线。
2. **大阳线验证**: B1 日前 20 交易日内存在 3~5 根涨幅 >= 5% 的大阳线，且换手率总和 <= 35%。
3. **整理区间识别**: 计算区间最高价（压力位）和最低价（支撑位）。
4. **B2 突破检测**: 收盘价 > 整理区间最高价 + 当日涨幅 >= 4%。
5. **放量确认**: 成交量 > 近10日均量 x 1.5。
6. **站稳多空线**: 突破后连续3天收盘均在多空线上方。
