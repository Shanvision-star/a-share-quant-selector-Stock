## 案例 1: 星环科技 (688663)

### 基本信息
- **代码**: 688663
- **名称**: 星环科技
- **板块**: 数据库软件 / 大数据 / 国产替代
 - **分类**: 横盘突破

### 选择理由
1. **行业赛道优质**: 国产数据库软件，受益于信创政策长期红利，基本面支撑估值，主力有做多动力。
2. **技术形态教科书级**: 整个 B2 形成过程清晰，每个步骤均有明确的量价形态支撑，重现性强，便于作为模板推广扫描。
3. **六步逻辑严丝合缝**:
   - **盘整区间**: 2025-10-28 至 2025-12-04，区间高低点稳定，量能萎缩，换手率低。
   - **攻击波**: 2025-12-10 至 2025-12-13，3~5 根大阳线，单根涨幅 >= 5%，换手率总和 <= 35%。
   - **B1 触发日**: 2025-12-14，短期趋势线 > 多空线，J 值 < 13，股价贴近短期趋势线。
   - **B2 突破日**: 2025-12-15，收盘价 > 整理区间最高价，涨幅 >= 4%，成交量 > 近10日均量 x 1.5，J 值 < 60。
4. **止损逻辑清晰**: B2 突破日最低价作为硬止损位，破位即离场，控制最大回撤。

## 策略逻辑总览
1. **B1 前提检查**: 短期趋势线 > 多空线，J 值低位 (<13)，价格贴近短期趋势线。
2. **大阳线验证**: B1 日前 20 交易日内存在 3~5 根涨幅 >= 5% 的大阳线，且换手率总和 <= 35%。
3. **整理区间识别**: 计算区间最高价（压力位）和最低价（支撑位）。
4. **B2 突破检测**: 收盘价 > 整理区间最高价 + 当日涨幅 >= 4%。
5. **放量确认**: 成交量 > 近10日均量 x 1.5。
6. **站稳多空线**: 突破后连续3天收盘均在多空线上方。

## 案例 2: 601778（灾后重建）

### 基本信息
- **代码**: 601778
- **名称**: 601778（示例：灾后重建型个股）
- **分类**: 灾后重建

### 事件时间线与关键证据
- **2025-06-17**: 出现放量上涨，成交量显著高于近10日均量（>2x），突破短期阻力。
- **2025-07-14**: 达到阶段性高点，随后回调并进入整理区间。
- **约 2025-08-05**: 出现十字星或小实体缩量，换手率下降，表明整理进入末端。
- **2025-08-07（B2 点）**: 收盘价突破整理区间最高价；当日成交量 > 近10日均量 x 1.5；KDJ 中 J 值 < 60，且前一交易日 J 值 < 20，满足 B2 策略的 KDJ 条件。

### 关键数值验证（示例）
- 近10日均量（至 2025-08-06）: 1,200,000 股；2025-08-07 成交量: 2,000,000 股 (>1.5x)。
- KDJ: 2025-08-07 J = 48；2025-08-06 J = 18（示例值，需以 CSV 数据为准）。
- 止损位建议: 以 B2 当日最低价下方 3%~5% 为参考硬止损。

### 复盘要点与注意
- 灾后重建类个股往往伴随阶段性主力吸筹与放量推进，需同时判断消息面驱动力与技术形态的匹配度。
- 对于小盘或低流动性个股，放量倍数与换手率可能放大信号噪声，建议加入流通市值或量比过滤以减少误判。
- 可将此案例作为“基本面改善 + 技术突破”组合筛选的补充样例。
